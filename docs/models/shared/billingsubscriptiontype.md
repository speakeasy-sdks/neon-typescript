# BillingSubscriptionType

Type of subscription to Neon Cloud.
Notice that for users without billing account this will be "UNKNOWN"



## Values

| Name                  | Value                 |
| --------------------- | --------------------- |
| `Unknown`             | UNKNOWN               |
| `Free`                | free                  |
| `Pro`                 | pro                   |
| `PlatformPartnership` | platform_partnership  |
| `Enterprise`          | enterprise            |
